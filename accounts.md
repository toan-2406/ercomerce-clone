# Tài khoản truy cập hệ thống CellphoneS Clone

Dưới đây là thông tin các tài khoản đã được đăng ký và kiểm tra thành công trên môi trường local.

## 1. Tài khoản User (Khách hàng)
- **Số điện thoại:** `0987654321`
- **Mật khẩu:** `password123`
- **Họ tên:** Antigravity User
- **Vai trò:** `customer`
- **Trạng thái:** Active

## 2. Tài khoản Admin (Quản trị viên)
- **Số điện thoại:** `0123456789`
- **Mật khẩu:** `adminpassword123`
- **Họ tên:** Admin System
- **Vai trò:** `admin`
- **Trạng thái:** Active
